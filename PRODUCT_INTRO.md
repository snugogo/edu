# 未来科普终端 - 产品介绍

## 产品概述

**未来科普终端** 是一款面向学校教育场景的 AI 智能终端设备软件，基于 Android 系统开发，实现硬件设备与License绑定，支持远程配置管理、主题内容下发、OTA升级等功能。

---

## 核心功能

### 1. 设备管理
- **硬件绑定**：设备首次联网自动注册，MAC地址唯一标识
- **License绑定**：设备自动绑定License，支持批量导入License
- **状态监控**：实时查看设备在线状态、最后活跃时间

### 2. License管理
- **批量生成**：自定义前缀+数量，快速生成License
- **批量导入**：支持粘贴License列表导入
- **状态追踪**：查看License绑定设备、SN、有效期

### 3. 主题内容管理
- **多主题支持**：创建不同主题，配置不同AI智能体
- **图片轮播**：主题下配置多张图片，终端自动轮播展示
- **一键下发**：批量绑定设备到指定主题

### 4. 数据分析
- **热门问题**：统计高频咨询问题
- **关键词分析**：提取用户关注热点
- **时段分布**：分析设备使用高峰时段

### 5. 紧急通知
- **即时推送**：向所有设备或指定设备发送通知
- **跑马灯展示**：终端屏幕滚动显示通知内容

### 6. OTA远程升级
- **版本管理**：上传新版本APK
- **强制更新**：可选是否强制更新
- **终端自动下载安装**

### 7. 双语支持
- **中英文切换**：后台支持中文/英文界面
- **多语言终端**：终端设备可配置显示语言

---

## 技术架构

| 层级 | 技术 |
|------|------|
| 终端 APP | Android Native (Java) |
| 后端服务 | Python Flask + SQLite |
| 管理后台 | HTML5 + JavaScript (SPA) |
| 存储 | 本地SQLite (可扩展OSS) |

---

## 部署方式

1. **服务器部署**：Flask服务部署到云服务器（推荐阿里云）
2. **终端配网**：设备连接网络后自动注册
3. **后台管理**：浏览器访问管理后台进行配置
