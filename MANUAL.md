## 1. 产品概述

**未来科普终端** 是一款基于 Android 系统的 AI 智能终端软件，主要应用于学校教育场景。

### 1.1 主要功能

- AI 语音交互
- License 硬件绑定
- 远程配置管理
- 主题内容下发
- OTA 远程升级
- 紧急通知推送
- 双语支持

## 2. 设备首次配置

### 2.1 启动终端

首次启动终端设备，显示主界面（待机状态）：

> **图2.1 主界面-待机状态**
> 
> - 背景：科技感壁纸
> - 顶部：显示"WIFI: ON"、"MIC: ON"状态
> - 中央：显示"待机中... 请喊 '小科小科' 唤醒我"

### 2.2 进入设置

**操作方法**：

1. 在主界面找到顶部左侧的 **管理员触发区**（隐藏区域）
2. 点击该区域 **5次**（快速点击）
3. 系统自动跳转到 **设置界面**

> **图2.2 设置界面入口**
> 
> 点击顶部左侧隐藏区域5次进入

### 2.3 配置 License 服务器

在设置界面填写服务器地址：

| 字段 | 说明 | 示例 |
|------|------|------|
| License 服务器地址 | 部署的服务器地址 | http://47.85.44.224:80 |

> **图2.3 配置服务器地址**
> 
> - 在"License 服务器地址"输入框填写服务器地址
> - 点击"保存配置"

## 4. 设置界面功能介绍

### 4.1 设备状态

设置界面顶部显示设备状态：

```
设备状态: 已注册 / 未注册
设备标识: dev_xxxxx
```

### 4.2 主要配置项

#### 4.2.1 License 服务器地址

- 填写服务器地址，用于设备注册和配置同步

#### 4.2.2 智能体 ID (AgentCode)

- 从阿里云 eAgent 平台获取
- 用于连接特定的 AI 智能体

#### 4.2.3 API 配置（可选）

| 字段 | 说明 |
|------|------|
| API Key | 从阿里云eAgent平台获取 |
| Product Key | 从阿里云eAgent平台获取 |

#### 4.2.4 License 配置（申请后填入）

| 字段 | 说明 |
|------|------|
| AppId | 从阿里云eAgent平台申请 |
| 设备序列号 SN | 从阿里云eAgent平台申请 |
| AppKey | 从阿里云eAgent平台申请 |

#### 4.2.5 主题配置

| 字段 | 说明 | 示例 |
|------|------|------|
| 主题名称 | 终端显示的名称 | 3F · 人工智能实验室 |
| 背景图片 | 终端背景（可选） | 选择本地图片 |

### 4.3 按钮操作

| 按钮 | 功能 |
|------|------|
| 注册设备 | 将设备注册到服务器并绑定 License |
| 保存配置 | 保存当前配置到本地和服务器 |

> 
> 包含：设备状态 → License服务器 → Agent ID → API配置 → License配置 → 主题配置 → 按钮组

## 6. 常见问题

### Q1: 设备无法注册？

**检查项**：
1. 网络是否正常
2. License 服务器地址是否正确
3. 服务器是否开启

### Q2: AI 无响应？

**检查项**：
1. 麦克风是否开启（MIC: ON）
2. Agent Code 是否配置正确
3. API Key 是否有效

### Q3: 如何更新 APK？

**方法**：
1. 在后台 OTA 管理上传新 APK
2. 终端自动检测并提示更新
3. 或在设置界面检查更新

### Q4: 如何修改主题？

**方法**：
1. 在后台创建/编辑主题
2. 在设备管理中绑定设备到主题
3. 终端自动同步
